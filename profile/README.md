# Scrivener Professional Writing Studio

## About Scrivener Software

Scrivener provides a complete writing environment for novels, screenplays, academic dissertations, and long-form non-fiction. Scrivener combines a powerful corkboard for visual outlining, an outliner for structural planning, and a full-screen composition mode for distraction-free writing. Scrivener allows writers to break manuscripts into fragments (scenes, chapters, sections) that can be rearranged at any time. Scrivener stores research materials — PDFs, images, web pages, notes — alongside the draft in a single binder. Scrivener supports scriptwriting formats (screenplay, stage play, radio play) with automatic formatting. Scrivener includes metadata for tracking character arcs, plot threads, chapter word counts, and revision status. Scrivener offers snapshot feature to save versions before major edits. Scrivener compiles to multiple formats: Word, PDF, ePub, Kindle, Final Draft, plain text, and markdown. Scrivener is trusted by bestselling novelists, Oscar-winning screenwriters, PhD candidates, journalists, and technical writers worldwide.

## Deep Dive — Writing Project Management

**Corkboard vs Outliner** — Corkboard displays index cards for visual scene arrangement; Outliner shows structural hierarchy with custom metadata columns. Choose based on whether you think spatially or hierarchically.

**Compile Settings** — The Compile feature transforms binder content into finished manuscript. Save custom compile presets for different outputs: ebook (ePub), print (PDF), submission (Word), or script (Final Draft). Learn to map binder folders to front/back matter.

**Snapshots vs Save As** — Snapshots capture a version of a single document (scene) without duplicating your project. Save As creates a completely new project. Use snapshots for experimental rewrites of one scene; use Save As for major structural overhauls.

**Metadata for Complex Projects** — Custom metadata fields track status (draft/revised/final), character point of view, plot thread (A/B/C), location, time period, or any dimension. Filter and view by metadata in Outliner.

**Research Integration** — Drag PDFs, images, audio files, and web archives into the Research folder. Open alongside draft in split view. Annotate PDFs and images directly within Scrivener.

## Quick Reference for Scrivener

| Operation | Scrivener Action |
|-----------|------------------|
| New document | Cmd+N (Mac) / Ctrl+N (Win) |
| New folder | Cmd+Shift+N (Mac) / Ctrl+Shift+N (Win) |
| Corkboard view | Cmd+2 (Mac) / Ctrl+2 (Win) |
| Outliner view | Cmd+3 (Mac) / Ctrl+3 (Win) |
| Compile | Cmd+E (Mac) / Ctrl+E (Win) |
| Take snapshot | Cmd+5 (Mac) / Ctrl+5 (Win) |
| Show inspector | Cmd+Option+I (Mac) / Ctrl+Alt+I (Win) |
| Full screen composition | Cmd+Shift+F (Mac) / Ctrl+Shift+F (Win) |
| Split editor vertically | Cmd+Option+V (Mac) / Ctrl+Alt+V (Win) |
| Insert annotation | Cmd+Shift+A (Mac) / Ctrl+Shift+A (Win) |
| Insert footnote | Cmd+Shift+F (Mac) / Ctrl+Shift+F (Win) |
| Add to research | Drag file to Research folder |
| Create collection | File > New > Collection |
